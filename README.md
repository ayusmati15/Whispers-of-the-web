# Whispers of the Web

A browser-based text distortion simulation inspired by the “Chinese whisper” concept.

## Description

Users submit sequential text “whispers,” which get distorted using different algorithms (blur, shuffle, symbol replacement). The application tracks the evolution of whispers in a **timeline** and dynamically updates **stats** for each algorithm.

## Features

- 3 distortion algorithms: **blur**, **shuffle**, **symbol replacement**
- Adjustable **distortion intensity** with a slider (1–10)
- Tracks **6+ sequential whispers** with dynamic timeline rendering
- Real-time **stats counter** showing algorithm usage
- Modularized **HTML, CSS, and JavaScript** for maintainability
- Smooth **CSS animations** for UX enhancements

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
